## 🔍 Key Design Decisions

### 1. Immutable ServiceContainer
**Why:** Prevents race conditions where services could be added/removed mid-request.

**Implementation:**
```python
@dataclass(frozen=True)
class ServiceContainer:
    datalake_gateway: Any = None
    # ... other fields
```

### 2. Fallback Logic for OMS Components
**Why:** Allows direct registration of managers (for testing/mocking) while maintaining automatic extraction from TradingContext.

**Implementation:**
```python
def get_order_manager() -> Any:
    container = get_container()
    
    # Direct registration takes precedence
    if container.order_manager is not None:
        return container.order_manager
    
    # Fall back to TradingContext
    ctx = container.trading_context
    if ctx is None:
        raise HTTPException(...)
    
    return ctx.order_manager
```

### 3. Actionable Error Messages
**Why:** Operators need to know what went wrong AND how to fix it.

**Before:**
```
"TradingContext not initialized. OMS is not available."
```

**After:**
```
"TradingContext not initialized. OMS is not available. 
To enable OMS, provide event_bus or trading_context 
when creating the FastAPI app."
```

### 4. Lifecycle State Tracking
**Why:** Prevents shutdown errors when lifecycle was never started.

**Implementation:**
```python
lifecycle_started = False
try:
    ctx.attach_lifecycle(lifecycle)
    lifecycle.start_all()
    lifecycle_started = True  # Track success
except Exception:
    # lifecycle_started remains False

# Shutdown
if lifecycle_started and lifecycle:
    lifecycle.stop_all()  # Only stop if started
```

## 🎓 Lessons Learned

1. **Immutable by Default:** Using `frozen=True` dataclasses prevents entire class of bugs
2. **State Tracking:** Always track initialization state for clean shutdown
3. **Actionable Errors:** Error messages should guide operators to resolution
4. **Test Isolation:** Global state requires careful test setup/teardown
5. **Fallback Patterns:** Direct registration → automatic extraction is flexible and testable
